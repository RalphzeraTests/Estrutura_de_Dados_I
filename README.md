# gtest

Repositório para testar o funcionamento do GTest

---

## Como Usar

### Para instalar as bibliotecado do GTest (Ubuntu)

- \$ sudo apt-get install libgtest-dev
- \$ sudo apt-get install cmake
- \$ cd /usr/src/gtest
- \$ sudo cmake CMakeLists.txt
- \$ sudo make
- \$ sudo cp \*.a /usr/lib
